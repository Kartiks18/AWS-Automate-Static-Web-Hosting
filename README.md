Project Title

Automating Static Website Deployment to Amazon S3 using Python (boto3)

Project Overview

This project demonstrates how to automate the deployment of a static website to Amazon S3 using the AWS SDK for Python (boto3).

Instead of manually uploading files through the AWS Console, a Python script automatically uploads website files such as HTML, CSS, JavaScript, and images to an S3 bucket.

This approach is commonly used in DevOps and Cloud Engineering to reduce manual work and ensure faster deployments.

Problem Statement

Uploading website files manually to Amazon S3 can:

Take time
Cause human errors
Be difficult to repeat consistently

The goal of this project is to automate the deployment process using Python and AWS SDK.

Connects to AWS S3
Reads website files from a local folder



Makes deployment faster and repeatable
AWS Services Used
Amazon Web Services
Amazon S3
boto3
IAM User
Access Keys
